# Tank Buff Reminder

**Tank Buff Reminder** is a lightweight, combat-safe World of Warcraft addon designed to ensure you never start a pull without your essential class buffs. It displays a draggable, resizable icon when important buffs are missing, acting as a secure button to cast the missing spell with a single click.

## Core Features
* **Intelligent Tracking**: Automatically detects your class and monitors essential buffs like Righteous Fury, Thorns, or specific Stance requirements.
* **Secure Casting**: The reminder icon functions as a secure action button, allowing you to cast the missing buff directly from the UI.
* **Combat Safe**: Handles combat lockdowns gracefully, ensuring the UI remains functional without "Action Blocked" errors.
* **Highly Customizable**: Includes a full options menu (`/tbr`) to toggle specific spells, adjust alert sounds, and resize or reposition the UI.
* **Visual & Audio Alerts**: Features a subtle pulse animation and optional sound cues to alert you when a buff drops.

## Supported Classes & Spells
The addon tracks the following based on your current class:
* **Paladin**: Righteous Fury, Devotion Aura.
* **Druid**: Thorns, Mark of the Wild (includes Gift of the Wild), Omen of Clarity.
* **Warrior**: Battle Shout, Commanding Shout, Defensive Stance.

## Configuration
Access the settings by typing `/tbr` in your chat or via the Interface Options menu.

* **Toggle Buffs**: Individually enable or disable tracking for any supported spell. Changes take effect instantly.
* **Sound Alerts**: Toggle the audio notification on or off.
* **Scaling**: Resize the icon (0.5x to 3.0x) using the grabber in the bottom-right corner.
* **Positioning**: Use **Shift + Left Click** to drag the icon anywhere on your screen.

## Installation
1. Download the repository.
2. Extract the `TankBuffReminder` folder into your `World of Warcraft/_classic_/Interface/AddOns/` directory.
3. Restart WoW or reload your UI (`/reload`).
